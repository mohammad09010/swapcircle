# SwapCircle (Book & Record Swap)

**Module:** Software Engineering (CMP-N204-0)  
**Team name:** SwapCircle  
**Team members:** Betab, Nitesh, Sagar, Arjun

## Project idea
SwapCircle is a community-first web app that helps people **swap books and vinyl records** for mutual benefit (not for profit). Users can list items they’re willing to exchange, browse items by category/tags, and request swaps.

## Tech stack (module tools)
- Frontend: HTML, CSS, JavaScript, **PUG**
- Backend: Node.js, Express.js
- Database: MySQL
- DevOps: Docker, Git, GitHub Actions (later sprints)
- Project management: GitHub Projects (Kanban)

## How to run (Docker)
### Prerequisites
- Docker Desktop installed

### Start
1. Copy `.env.example` to `.env` (optional — compose already includes defaults)
2. Run:

   ```bash
   docker compose up --build
```
